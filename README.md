# challenge-practica-TelecomX_2
proyecto final de telecom
TelecomX – Predicción de Cancelación de Clientes (Churn)
Este proyecto analiza el comportamiento de los clientes de TelecomX para identificar los factores asociados a la cancelación del servicio (churn) y desarrollar modelos de Machine Learning capaces de predecir qué clientes tienen mayor probabilidad de abandonar la empresa.

El objetivo es generar información útil que permita a la empresa anticiparse a la pérdida de clientes y diseñar estrategias de retención más efectivas.

Fuente del dataset:
